## 🧭 Resume Project Description

> **Smart AI Expense Tracker** — A modern, web-based personal finance dashboard built with HTML, CSS, and Vanilla JavaScript. Implements local storage persistence, category analytics using Chart.js, budget management, and AI-powered spending insights. Designed with glassmorphism UI, responsive layout, and smooth animations for startup-level user experience.

---

## 💡 Hackathon Presentation Points

1. **Problem:** Lack of smart, lightweight personal expense apps with AI insights.  
2. **Solution:** Built a predictive and insightful expense tracker using only front-end technologies.  
3. **Innovation:** AI-style insights and gamification (badges, warnings).  
4. **UI/UX:** Glassmorphic design, responsive layout, animated counters, charts.  
5. **Tech Simplicity:** 100% Vanilla JS — no frameworks, easy to deploy.  
6. **Scalability:** Could add APIs, authentication, or AI service later.  
7. **Impact:** Helps users build mindful spending habits.
